# Video Face Reference Builder

## 一、解决的核心问题

**输入约束**：视频中人脸始终不完整——极度近距离拍摄、随机角度、运动模糊、手部遮挡，没有任何单帧包含完整正脸。

**目标输出**：该人的完整清晰正脸肖像。

**现实场景**：监控录像、手机随拍、社媒短视频——这类"非配合"视频大量存在，现有工具都要求至少有一帧完整人脸，无法处理这种情况。

---

## 二、技术管线

```
原始视频
   │
   ▼
① 帧提取 + 质量初筛          → 过滤运动模糊/曝光异常帧
   │
   ▼
② 人脸关键点检测 + 区域标注   → 判断每帧露出了哪个区域（眼/鼻/嘴/颧骨...）
   │
   ▼
③ 分区最优帧选择              → 每个人脸区域找质量最好的帧
   │
   ▼
④ 可选单帧增强对照            → 只增强已选帧，成功图作参考，失败帧继续用原图
   │
   ▼
⑤ 跨帧人脸区域对齐 + 拼合     → 几何对齐，将各区域拼到统一坐标系
   │
   ▼
⑥ 缺失区域补全 + 全局融合     → 填补没有任何帧覆盖的区域，消除拼接痕迹
   │
   ▼
完整正脸重建结果
```

**当前瓶颈**：缺失区域补全容易产生幻觉，嘴部/人中等区域需要手动修正，是从 Demo 走向自动化工具的关键门槛。

---

## 三、各环节技术选型

| 环节 | 方案 | 理由 |
|------|------|------|
| 帧质量评分 | 自研算法（清晰度 + 人脸面积 + 角度评分） | 规则明确，AI 反而过重 |
| 单帧增强对照 | CodeFormer / GFPGAN | 可选辅助，不替换原始关键帧 |
| 区域对齐拼接 | 自研算法（特征点对齐 + 泊松融合） | 几何问题，算法更可控、更准确 |
| 缺失区域补全 | AI（扩散模型 inpainting） | 纯算法无法凭空生成未见区域 |
| 身份一致性校验 | AI（人脸识别模型打分） | 防止生成了"另一个人的脸" |

**核心设计原则**：已有区域用算法对齐拼合（精准可控），真正缺失的区域用受约束的 AI 补全（以身份相似度作为约束，抑制幻觉）。

---

## 四、创新点

1. **跨帧区域聚合**：现有开源工具处理单帧，本项目处理"无任何完整帧"的视频
2. **受约束的缺失补全**：AI 补全 + 身份一致性反馈，解决纯生成的幻觉问题
3. **端到端管线**：从原始视频到证件照级肖像，目标无人工干预

---

## 五、对比现有开源项目

| 项目 | 能力 | 与本项目差异 |
|------|------|------------|
| CodeFormer | 单帧盲目人脸修复 | 仅作可选对照；近距离局部脸检测不稳定 |
| CFR-GAN | 遮挡下单图 3D 重建 | 单张图，非跨帧聚合 |
| VideoFaceExtractor | 从视频提取人脸帧 | 只提取，不重建 |
| GANFit | 3DMM 拟合 | 需要清晰输入图 |
